### Overview

Kimigayo OS is a lightweight, fast, and secure container-focused operating system that inherits the design philosophy of Alpine Linux. Designed to run as Docker images, it aims to operate with minimal resources and deliver high performance in container environments and microservice architectures.

### âœ¨ Key Features

- ğŸª¶ **Lightweight**: Base image under 5MB
- âš¡ **Fast**: Boot time under 10 seconds, low memory consumption (128MB minimum)
- ğŸ”’ **Secure**: Secure-by-default design with comprehensive compile-time and runtime hardening
- ğŸ§© **Modular**: Select only needed features, GUI modules can be added
- ğŸ”„ **Reproducible Builds**: Guaranteed bit-identical build outputs
- ğŸŒ **Multi-Architecture**: Supports x86_64 and ARM64 (RISC-V planned)
